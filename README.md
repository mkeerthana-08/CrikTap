CrikTap is a browser-based Hand Cricket game where you can experience the thrill of batting and bowling in one match. The game is designed with animations, interactive score updates, and colorful gradient backgrounds to provide a fun, engaging cricket experience.

## Features
- Choose to BAT or BOWL first
- Play both batting and bowling in a single match
- Dynamic scoring with OUT rule
- Animated results (Win, Lose, Draw)
- Confetti animation for victories
- Play Again option
- Yellow-themed interface with gradient background
- Fully interactive and beginner-friendly

## How to Play
1. Open `index.html` in your browser (Live Server recommended)
2. Choose **BAT** or **BOWL**
3. Click numbers 1‚Äì6 to play each turn
4. Matching numbers = OUT
5. Role switches automatically after first OUT
6. Second OUT ends the match, displaying the final result
7. Click **Play Again** to restart

## Technologies Used
- HTML5
- CSS3 (Animations & Background Effects)
- JavaScript (Game Logic)
